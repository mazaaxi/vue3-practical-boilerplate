# vue3-practical-boilerplate

## Project setup
```
yarn install
```

## Try to run the application

Run the application.

```
yarn serve
```

Run a different terminal from the above and run the API server.

```
yarn serve:api
```

Access the following URL from your browser.

```
http://localhost:5040
```
